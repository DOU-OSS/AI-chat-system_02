# AI-chat-system_02
这是一个AI对话系统，实现用户与AI的智能对话功能。

# 基础功能

1. **用户功能**：实现用户登录注册功能
2. **AI对话功能**：按照业界标准支持接入不同AI模型
3. **角色系统**：能够为AI创建角色并管理
4. **对话保持**：支持同一个对话保留对话信息
5. **记忆系统**：实现简单的记忆功能（上下文窗口中短期记忆）

# 相关技术
采用 **前后端分离技术栈** 开发，核心技术简介如下：  
**后端**：以 Spring Boot 为基础，结合 Spring Security + JWT 实现用户认证授权，通过 BCrypt 加密保护密码、CORS 解决跨域问题，搭配 WebSocket 支撑实时聊天  

**前端**：基于 Vue + Vite 构建交互界面，通过 Axios 实现前后端通信  

**数据库**：使用关系型数据库存储用户/聊天数据，辅以 Swagger 调试接口，整体覆盖“认证授权、实时通信、前后端协同”核心需求
